# Security Policy

This repository is a public-safe reference kit for compute delivery receipts.

It must not contain production LUNQ exchange, redemption, settlement, wallet, liquidity, custody, signer, KYC, KYB, sanctions screening, wallet-risk, or pricing logic.

## Do Not Disclose Publicly

- API keys, private keys, seed phrases, signatures, session tokens, or webhook secrets
- private customer, supplier, invoice, contract, pricing, or workload data
- production detector payloads with private hardware identifiers
- production wallet, exchange, custody, market-making, or token conversion code
- compliance vendor configuration or internal screening rules
- claims about guaranteed token value, yield, ROI, listings, futures products, or supplier matching

## Reporting

If you find sensitive data in this repository, open a GitHub security advisory or contact the repository owner privately. Do not create a public issue with the secret value.
